# V2Press

## About

V2Press is a WordPress theme which will turn a regular WordPress (blogging) site into a community or a forum. V2Press is very lite and clean.

## Feature

-   Deeply integrated with WordPress
-   Built-in use of HTML5 and CSS3
-   Clean UI
-   Minimal features
